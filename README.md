# Code Vault Personal

> **Disclaimer:**
> This codebase is **not production-level**. It is intended strictly for **personal use**, primarily to share code between my university PC and other personal devices. It may be developed into a production-level application at a later time.

The live application can be accessed here: [https://snippet300809.web.app/](https://snippet300809.web.app/)

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the application locally:
   `npm run dev`
